# Esar Cars API Documentation

## Overview

Esar Cars is a car rental platform API built with Laravel.

The platform allows users to:
- Browse cars
- Rent vehicles
- Manage cars as partners
- Process bookings


## Backend Stack

- Laravel 12
- MySQL
- Laravel Sanctum
- Spatie Laravel Permission
- REST API


## Authentication

The API uses Token Based Authentication.

Authenticated requests must include:

Authorization: Bearer TOKEN
